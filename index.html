  • O que é HTML

    • HTML não é uma linguagem de programação como todos pensam, na verdade o termo HTML significa "Hyper Text Markup Language" em português significa "Língua de Marcação de Hiper Texto".

    O HTML é apenas um marcador de hipertexto que pode separar o site em itens e tópicos, redirecionar links e definir a exibição do site.

  • Como o HTML funciona

    • O HTML funciona com blocos de tags, que definem a marcação do texto, se ele é um parágrafo, se altera fonte do texto, se deixa o texto em itálico ou negrito, se possui algum link âncorado naquele texto e etc.

  • Como o HTML é estruturado

    • O HTML é estruturado em forma de bloclos de tags, por exemplo para inicar um texto como esse você deve por um "<p>" no início do texto e um "</p>" ao final do texto, isso criará um parágrafo na página e tudo que estiver escrito dentro desse bloco de tag "<p>" estará impresso na tela.

    existem também algumas tags que servem para modificar o estilo do texto como "<strong>", que deixa o texto dentro de seu bloco em negrito, "<i>", deixa tudo dentro de seu bloco em itálico, "<u>", deixa tudo dentro de seu bloco com sublinhado e as tags "<h1>", "<h2>", "<h3>", "<h4>", "<h5>" e "<h6>", que alteram o tamanho do texto afim de adicionar títulos e etc.

    existem várias tags diferentes para um bom funcionamento de uma página web.
